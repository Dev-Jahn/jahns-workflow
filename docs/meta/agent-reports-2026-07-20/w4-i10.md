## 판정

- `docs/invariants.md:26`의 I-10은 모델 입력을 목표·경계·판단 요청으로 제한하고 bookkeeping protocol 전달을 금지한다.
- `docs/adr/ADR-0014-m1a-acceptance-basis.md:145-148`은 허용 예외를 `WAYSTONE_REPORT` 보고 계약 stanza로 한정하고 `registry(tasks.yaml)` 및 `round`를 금지 표면으로 명시한다.
- 허용 stanza는 `templates/delegate-prompt.md:26-42`의 `## Report (required)`이다. 이 판정은 stanza 밖 `WAYSTONE_REPORT` 참조의 허용 여부 같은 해석에 의존하지 않는다.
- `scripts/common.py:23,1000-1002`는 registry source가 `tasks.yaml`임을 고정한다. `scripts/delegate.py:544-592`는 그 registry에서 `status`, `milestone`, `round`, dependency status, `anchor`, `notes`를 packet에 담는다.
- `scripts/delegate.py:651-665`는 해당 상태를 prompt-visible 문자열로 만들고, `:673-687`은 이를 template의 `{{TASK_BLOCK}}`에 치환한다. 직접 render에서 `- round: 2026-07-20-probe`가 관측됐으므로 §5의 명시 금지와 직접 충돌한다. 함께 관측된 표면은 `status`, `milestone`, `anchor`, `routing_note`다. dependency가 있으면 `deps`와 dependency status도 같은 함수가 렌더한다.
- 이는 helper-only 현상이 아니다. `scripts/delegate.py:2156-2166`은 rendered 결과를 `prompt.txt`로 저장해 implementer transport에 전달하고, Codex `:1632-1635`와 Claude `:1712-1717` 경로 모두 그 파일을 모델 stdin으로 사용한다.
- 원본 `templates/delegate-prompt.md`에는 `round` literal이 없지만, 계약 대상인 실제 조립 결과에는 존재한다. 따라서 현행 템플릿 조립 경계가 이미 위반 상태이며 acceptance 1(현행 template에서 green)을 만족시키는 정직한 부정형 테스트를 추가할 수 없다.

## 재현 명령 원문

```bash
uv run python - <<'PY'
import sys
from pathlib import Path

sys.path.insert(0, "scripts")
import delegate

data = {
    "project": "demo",
    "tasks": [{
        "id": "feat/i10-probe",
        "title": "implement the requested goal",
        "status": "active",
        "milestone": "M1-A",
        "round": "2026-07-20-probe",
        "anchor": "src/example.py:10",
        "notes": "stay inside the declared bound",
        "scope": ["src/example.py"],
        "accept": ["an independent verifier judges criterion alpha"],
    }],
}
packet, _ = delegate._build_packet(
    data,
    "feat/i10-probe",
    [],
    Path.cwd(),
    routing_note="budget favors direct execution",
)
prompt = delegate._render_prompt(packet, "a" * 40)
prohibited_observations = (
    "- status: active",
    "- milestone: M1-A",
    "- round: 2026-07-20-probe",
    "- anchor: src/example.py:10",
    "- routing_note: budget favors direct execution",
)
for observation in prohibited_observations:
    assert observation in prompt, observation
    print(observation)
print("rendered prompt contains registry/round/routing bookkeeping surfaces")
PY
```
